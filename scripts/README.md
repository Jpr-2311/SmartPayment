# FinPilot AI - Setup Scripts

This directory contains automation scripts for project setup and development.

## Scripts

| Script | Platform | Purpose |
|---|---|---|
| `setup.ps1` | Windows (PowerShell) | Full project setup automation |
| `setup.sh` | Linux/macOS (Bash) | Full project setup automation |

## Usage

### Windows
```powershell
.\scripts\setup.ps1
```

### Linux/macOS
```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```
